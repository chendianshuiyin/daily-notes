# 每日记录助手 - 项目概述

## 项目简介

**每日记录助手** 是一款基于 Flutter 开发的 Android 手机应用，旨在帮助用户便捷地记录每天的活动、想法和待办事项，并提供归档总结功能。应用采用本地优先的数据存储策略，同时支持通过 WebDAV 协议进行云端同步。

## 核心价值

- **简洁高效**：流畅的记录体验，让用户专注于内容本身
- **数据安全**：本地存储优先，用户完全掌控数据
- **灵活同步**：WebDAV 支持，兼容多种云存储服务
- **可视化统计**：日历热力图展示记录频率，激励持续记录

## 目标用户

- 希望养成每日记录习惯的用户
- 注重数据隐私和自主权的用户
- 需要回顾和总结日常活动的职场人士
- 喜欢使用自建云存储服务的技术爱好者

## 主要功能模块

| 模块 | 核心功能 |
|------|----------|
| 今日记录 | 富文本编辑、日期选择、时间戳插入、保存记录 |
| 历史记录 | 记录列表、搜索过滤、标签分类、归档管理 |
| 统计总结 | 日历热力图、连续打卡统计、月度概览、每日详情 |
| 设置管理 | WebDAV 配置、主题切换、数据管理 |

## 技术栈概览

- **框架**: Flutter 3.10+
- **目标平台**: Android (初期), 可扩展至 iOS
- **本地存储**: SQLite + Hive/SharedPreferences
- **网络同步**: WebDAV 协议
- **UI 设计**: Material Design 3，支持深色/浅色主题

## 项目范围

### 第一阶段 (MVP)
- 基础记录功能（创建、编辑、删除）
- 本地数据存储
- 历史记录浏览
- 基础统计展示

### 第二阶段
- WebDAV 同步功能
- 标签系统
- 搜索功能
- 深色主题

### 第三阶段
- 数据导出/导入
- 提醒通知
- 小组件支持
- 性能优化
