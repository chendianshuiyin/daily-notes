# UI 设计规范

## 1. 设计理念

**"秩序与活力"**
- **秩序**：通过卡片式布局和清晰的排版减少认知负荷。
- **活力**：使用醒目的强调色（蓝色）和微交互（如热力图动画）增加记录的趣味性。

## 2. 色彩系统 (Color Palette)

基于 Tailwind CSS 的配置进行定义，支持深色模式。

### 核心色
| 角色 | 颜色值 (Light) | 颜色值 (Dark) | 用途 |
|------|---------------|---------------|------|
| **Primary** | `#135bec` (Blue) | `#135bec` | 主要按钮、强调文字、图标、热力图高亮 |
| **Danger** | `#ef4444` (Red) | `#ef4444` | 删除操作、错误提示 |
| **Success** | `#22c55e` (Green) | `#22c55e` | 同步成功、完成状态 |

### 背景与表面
| 角色 | 颜色值 (Light) | 颜色值 (Dark) | 用途 |
|------|---------------|---------------|------|
| **Background** | `#f6f6f8` | `#101622` / `#111722` | 应用底层背景 |
| **Surface** | `#ffffff` | `#1d2532` / `#1e2736` | 卡片、弹窗、底部导航栏 |
| **Border** | `#e2e8f0` | `#2d3748` | 分割线、边框 |

### 文本
| 角色 | 颜色值 (Light) | 颜色值 (Dark) | 用途 |
|------|---------------|---------------|------|
| **Primary Text** | `#0f172a` (Slate-900) | `#ffffff` | 标题、主要内容 |
| **Secondary Text** | `#64748b` (Slate-500) | `#94a3b8` (Slate-400) | 辅助说明、次要信息 |
| **Caption** | `#94a3b8` | `#64748b` | 微小标签、占位符 |

## 3. 字体排印 (Typography)

- **字体家族**：
  - 英文/数字：`Manrope` (Google Fonts) - 具有现代感和良好的可读性。
  - 中文：`Noto Sans SC` (思源黑体) - Android 系统默认，保证兼容性。

- **层级规模**：
  - **H1 (页面标题)**: 20px / Bold
  - **H2 (卡片标题)**: 18px / Bold
  - **H3 (正文强调)**: 16px / SemiBold
  - **Body (正文)**: 14px / Regular (行高 1.5)
  - **Caption (说明)**: 12px / Medium

## 4. 图标系统

- **图标库**：Material Symbols Outlined
- **风格**：
  - 默认状态：线性 (Outlined)
  - 选中/激活状态：填充 (Filled) - *可选，或者保持统一线性风格加粗*

## 5. 组件规范

### 5.1 卡片 (Card)
- **圆角 (Radius)**: `12px` (≈ xl) 或 `16px` (≈ 2xl)
- **阴影 (Shadow)**:
  - Light: `shadow-sm` (轻微浮起)
  - Dark: 无阴影，依赖背景色差 (`bg-surface-dark`) 和边框 (`border-gray-800`) 区分层级。
- **内边距 (Padding)**: `16px`

### 5.2 按钮 (Button)
- **主按钮 (Primary)**:
  - 高度: `56px` (大按钮)
  - 圆角: `12px`
  - 效果: 悬浮/按下时有缩放 (`scale-95`) 或 透明度变化。
- **图标按钮 (Icon Button)**:
  - 尺寸: `40x40`
  - 圆角: `Full` (圆形)

### 5.3 热力图 (Heatmap)
- **单元格**: 正方形，圆角 `4px` - `8px`。
- **颜色等级**:
  - Level 0 (0): Gray-200 / Gray-800
  - Level 1 (1-2): Primary / 30%
  - Level 2 (3-5): Primary / 60%
  - Level 3 (6-8): Primary / 80%
  - Level 4 (9+): Primary / 100%

## 6. 交互动效

- **页面切换**: 推荐使用 Flutter 默认的 `Cupertino` (iOS风格) 滑动切换或 `FadeThrough` 渐隐渐现，保持轻盈感。
- **列表滚动**: 增加 `Overscroll` 弹性效果。
- **点击反馈**: 所有可交互元素必须有 InkWell 水波纹或缩放反馈。
