# Flutter 开发指南 (Index)

本指南旨在为 Google Antigravity 提供一套全面、结构化且符合现代最佳实践的 Flutter 开发规范。原始 `rule.md` 的所有内容已按逻辑分类拆分为以下章节，以便于查阅和维护。

## 章节索引

### 1. [通用原则与项目结构](./guidelines/01_general_principles.md)
> 涵盖交互准则、项目标准结构、Flutter 基本样式导引以及包管理 (Pub) 规范。

### 2. [代码质量与最佳实践](./guidelines/02_code_quality_and_best_practices.md)
> 包含命名约定、代码简洁性、Dart 语言高级特性（空安全、异步等）以及 Flutter Widget 开发的最佳实践。

### 3. [架构、数据与 API 设计](./guidelines/03_architecture_and_data.md)
> 讨论 MVC/MVVM 层级划分、基于功能的组织方式、数据序列化 (JSON) 以及 API 设计哲学。

### 4. [开发功能：状态管理、路由与日志](./guidelines/04_development_features.md)
> 详细说明内置状态管理方案、`go_router` 导航配置、结构化日志记录以及代码生成流程。

### 5. [测试、设计与无障碍](./guidelines/05_testing_and_design.md)
> 涵盖三层测试策略、视觉设计细节（阴影、排版）、资源管理以及无障碍 (A11Y) 适配要求。
